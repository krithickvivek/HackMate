A platform for hackathon team formation with role-based access, skill matching, and secure authentication.

## Problem Statement
Students often struggle to find suitable teams for hackathons, while teams face difficulty finding skilled members. This leads to missed opportunities and weaker participation.

## Features
- User registration and login
- Role-based access (Team Leader / Member)
- Team creation and management
- Post and apply for openings
- Skill-based search
- Secure authentication
- Responsive UI

## Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## How to Run the Project

1. Clone the repository
   git clone https://github.com/yourusername/HackMate.git

2. Install dependencies
   cd backend
   npm install
   cd ../frontend
   npm install

3. Start backend
   npm start

4. Start frontend
   npm start
